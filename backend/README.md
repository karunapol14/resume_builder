# Backend - Resume Builder

This folder contains the Express backend, database schema and instructions.

Setup

1. Copy `.env.example` to `.env` and fill values.
2. Install dependencies:

```powershell
cd backend
npm install
```

3. Create Postgres database and run schema in `schema.sql`.

4. Start server:

```powershell
npm run dev
```

Files
- `index.js`: main server
- `schema.sql`: database schema
